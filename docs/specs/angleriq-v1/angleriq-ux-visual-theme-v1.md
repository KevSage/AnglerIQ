# AnglerIQ UX Visual Theme — V1

Status: Finalized  
Owner: Kevin Sage  
Theme: Sports Hybrid + Mobile First + Light/Dark  
Version: 1.0

---

## 1. Visual Philosophy

AnglerIQ blends:

- **Sports performance UI**: bold stat blocks, sharp layout, energetic accents
- **Premium tech minimalism**: clean spacing, restrained typography, simplified forms
- **Angler familiarity**: natural greens, water tones, subtle textures

The result is a UI that feels like:

> “A sports analytics app for anglers.”

## 3. Typography

**Font Family:**

- Primary: **Inter** (UI, legible, modern)
- Secondary: **Roboto Condensed** (sports metrics)

### Hierarchy

- H1: 28px / 700
- H2: 22px / 600
- H3: 18px / 600
- Body: 16px / 400
- Label: 14px / 500
- Stat Number: 24px / 700 (Roboto Condensed)

---

## 4. Iconography

Rounded geometric icons  
consistent with:

- fishing (depth, lure, action)
- analytics (graphs, stats)
- mobile-first readability

---

## 5. Layout System

### Mobile-First Grid

- 4pt spacing
- 8/12pt vertical rhythm
- Edge padding: 16–20px
- Card radius: 16px
- Bottom nav height: 64px

### Atmosphere

- Light mode: clean, bright, crisp
- Dark mode: deep charcoal, underwater blues
- Gradients: subtle vertical fades for hero areas

---

## 6. Motion / Interactions

Lightweight sports micro-interactions:

- Card hover raise
- Tap ripple
- Slide-in animations for results
- Smooth transitions for theme switching
