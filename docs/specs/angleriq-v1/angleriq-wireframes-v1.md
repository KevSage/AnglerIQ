# AnglerIQ Wireframes — V1

Version: 1.0  
Device Base: iPhone 13 / Android Medium  
Orientation: Portrait-first

---

## 1. Home Screen (Light + Dark)

### Layout

- Hero banner: “Dial in the Perfect Fishing Pattern”
- Start Button (primary.sage)
- Quick Input Tiles:
  - Season
  - Water Temp
  - Water Clarity
  - Location
- Performance Cards (“Today’s Conditions”, “Best Windows”)
- Bottom Navigation:
  - Home
  - Pro
  - History
  - Settings

### UX Notes

- Hero gradient (light: white→sage tint / dark: navy→charcoal)
- Quick inputs always visible without scrolling
- Bottom nav sticky

---

## 2. Input Screen

### Fields (scroll)

- Water Temp (slider or numeric)
- Water Clarity (select)
- Season / Month (select)
- Weather (optional)
- Time of Day
- Location (optional)

### CTA

- “Generate Pattern” button (sports style)

### Validation

- Minimum required: Temp + Month

## 3. Basic Results Screen

### Layout

- Phase Badge (Pre-Spawn / Spawn / etc.)
- Depth Zone Chip

- Secondary Techniques

- Quick Tips
- Upgrade CTA for Pro (“Unlock Full Pattern”)

Clean, minimal, not overwhelming.
